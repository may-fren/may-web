# MAY Web

MAY yönetim paneli web uygulaması.

## Teknolojiler

- React 19 + TypeScript
- Vite 7
- Ant Design 6
- React Router 7
- Axios

## Geliştirme

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up -d
```

## Port

- **8091** — Nginx (production)
- **5173** — Vite dev server (development)
